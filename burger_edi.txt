Merci pour votre commande d'essai.
J'ai remarqué les points suivants:
Le segment NAD + BY doit contenir le numéro de client:
NAD + BY + <ILN> :: 9 ++ CUISINES TRANSTEC BF ++ 12 avenue des prés - - + Montigny-le-bretonneux ++ 78180 + '

NAD + BY + <numéro client chez le fournisseur> :: 91 ++ TRANSTEC CUISINES BF ++ 12 avenue des prés - - + Montigny-le-bretonneux ++ 78180 + '

Dans votre exemple, NAD + BY + 10008 :: 9 ++ TRANSTEC CUISINES BF ++ 12 avenue des prés - - + Montigny-le-bretonneux ++ 78180 + 'a été transmis, où 10008 n'est pas un ILN valide.

Les variantes 402, 700 et 750 doivent être transférées à chaque commande
Les attaques doivent être transmises comme suit:
PIA + 1 + <L / R / M>: 6 :: 91 '

Dans votre exemple, PIA + 1 + J: 6 :: 91 'a été transmis

La profondeur d'une armoire doit toujours être transmise dans le fichier EGI sans l'épaisseur de la façade:
Mesure_T = 575,00 => Mesure_T = 555,00

Puis j'ai remarqué que vous n'aviez pas commandé avec un catalogue actuel. Les articles ECO ... etc. ne sont plus inclus dans le nouveau catalogue 2020.

Je suis heureux de répondre à toute question supplémentaire.
